# Welcome to the RoleModels Prototyping Tool Playground

The Fluid Framework is a real-time collaboration technology that can be used to sync state between multiple users.
The playground provides a way to interact with the prototyping tool demos running side by side in the browser.

In the demos I focused on the cross-device interactions therefore I did not include a mechanism to change or add roles.
There are two pre-defined roles (capturer and viewer) but the tool is supports any other roles.
When viewing a demo you can perform the following actions:

- View a demo on its own by clicking the **Canvas tab**.
- Get details about a demo by clicking the **Docs tab**.

**To get started** select a demo from the left sidebar!

# Additional Info

The roles are automatically distributed when devices connect. By clicking the header buttons you can change the role of the current device.

Each combined view includes a button "E" that can be used to see the UI and edit it.

The available components are:
